#  PixlCut
## is a lightweight core PHP based image manipulation web app

## features:
- Stable (based on PHP GDImage extension)
- Lightweight
- Simple & Easy to use
- Low Maintenance 
- Open Source (duh!!)

## Requirements:
### PHP 7.2 & Up
### gdextension enabled
